# Nestor v1.0.0

This is the first public release of Nestor — an AI-powered file manager for Windows that lets you organize your files through a natural language chat interface.

---

## Installation

Download `Nestor Setup 1.0.0.exe` below and run it. The installer guides you through setup in under a minute.

**This is the only download you will ever need.** Nestor updates itself automatically in the background whenever a new version is released. You do not need to download a new installer for future updates — the app checks for updates on launch and installs them silently.

---

## What's included in this release

**AI Chat for File Management**
Describe what you want in plain language. Nestor analyzes your folder, proposes a plan, and executes file operations only after your approval.

**Local AI — fully offline and private**
Nestor runs on your device using Ollama with `llama3.2:3b`. No files or folder names are sent to any server. Alternatively, connect any OpenAI-compatible API in Settings.

**File Explorer with drag-and-drop context**
Browse your folder tree, drag files into the chat to give the AI context, or type `@filename` to attach files to a message.

**Action Review Panel**
Every proposed action (move, rename, delete, create folder) is shown for review before anything is executed. Nothing changes on disk without your confirmation.

**Activity Log**
A complete history of everything Nestor has done, with timestamps.

**Light and dark theme, six accent colors**
Customize the look from Settings.

**Guided onboarding**
First launch walks you through choosing local or API mode and selecting a working folder.

---

## System Requirements

Windows 10 / 11 (64-bit) · 4 GB RAM minimum · [Ollama](https://ollama.com) for local AI mode

---

## A note on auto-updates

Once installed, Nestor checks for new releases on GitHub automatically. Updates download and install in the background — you will see a notification when a new version is ready. There is no need to visit this page again or download a new installer.
